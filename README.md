## 🛠️ 如何快速开始？

### 1. 环境准备
确保你的电脑已经安装了 **Node.js** 环境。

### 2. 部署步骤
1.  **克隆/下载** 仓库到本地。
2.  **安装项目依赖**：
    ```bash
    npm install
    ```
3.  **配置 API Key**：
    在根目录找到 `.env.local` 文件，填入你的 `GEMINI_API_KEY`。
4.  **启动预览**：
    ```bash
    npm run dev
    ```
