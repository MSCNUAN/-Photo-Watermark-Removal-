✨ AI 照片去水印工具 (Photo Watermark Removal)
🚀 一个基于 Gemini AI 技术的高效、智能照片去水印方案。

本工具专为小白设计，通过 AI 算法自动识别并抹除图片中的各种水印、杂物或路人。无需复杂的 PS 操作，一键还原纯净画面！

🌟 项目亮点
智能识别：深度集成 Gemini AI，精准识别背景与瑕疵。

简单易用：界面直观，真正的“小白向”操作流程。

灵活部署：支持本地 Node.js 环境快速启动。

完全开源：基于 AI Studio 模板开发，欢迎折腾分享。

🛠️ 如何快速开始？
1. 环境准备
确保你的电脑已经安装了 Node.js 环境。

2. 部署步骤
克隆/下载仓库到本地。

安装项目依赖：

Bash
npm install
配置 API Key： 在根目录找到 .env.local 文件，填入你的 GEMINI_API_KEY。

启动预览：

Bash
npm run dev
3. 在线访问
不想配置环境？直接通过 AI Studio 访问： 👉 点击此处在线体验
